# Horse Racing Management System - Backend

## Description

Horse Racing Management System is a backend system for managing horse racing tournaments, horses, jockeys, referees, spectators, race results, rankings, predictions, prizes, and user roles.

## Main Actors

* Admin
* Horse Owner
* Jockey
* Race Referee
* Spectator

## Main Features

### Horse Owner

* Register account
* Register horse for tournament
* Manage horse information
* Select jockey for horse
* View race schedule
* View race result, ranking, and prize

### Jockey

* Register jockey account
* Accept or reject race invitation
* View assigned races
* View personal results and achievements

### Race Referee

* Check horse information before race
* Monitor race
* Record violations
* Confirm race result
* Create referee report

### Spectator

* View tournament and race schedule
* Follow live results and rankings
* Predict race result
* Receive prediction reward notification

### Admin

* Manage users and roles
* Manage tournaments and races
* Approve registrations
* Manage horses and jockeys
* Assign referees
* Publish race results
* Manage predictions
